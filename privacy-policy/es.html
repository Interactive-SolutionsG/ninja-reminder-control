Política de Privacidad - Ninja Reminder Control

Última actualización: Mayo 2026

Versión: 1.0

1. Información General

Ninja Reminder Control es una aplicación de calendario y recordatorios desarrollada con el máximo respeto a tu privacidad. Esta política explica qué información recopilamos (o más bien, qué NO recopilamos) y cómo protegemos tu privacidad.

Desarrollador: Interactive Solutions

Contacto: interactive.sup.solutions@outlook.com

2. Compromiso de Privacidad

Principio Fundamental:

Ninja Reminder Control NO recopila, NO almacena, NO transmite y NO comparte ningún dato personal.

3. Datos que NO Recopilamos

Esta aplicación NO recopila ninguno de los siguientes datos:

Datos Personales:

  • Nombre, apellidos, dirección
  • Correo electrónico
  • Número de teléfono
  • Fecha de nacimiento
  • Ubicación geográfica
  • Dirección IP

Datos de Uso:

  • Estadísticas de uso de la aplicación
  • Telemetría o análisis de comportamiento
  • Horarios de uso
  • Frecuencia de uso

Datos Técnicos:

  • Información del dispositivo
  • Sistema operativo
  • Especificaciones de hardware
  • Identificadores únicos del dispositivo
  • Logs de errores o diagnóstico

Nota sobre identificadores de dispositivo:

Microsoft Store utiliza identificadores de dispositivo para gestionar licencias entre tus dispositivos. Sin embargo, nosotros como desarrolladores NO tenemos acceso a estos identificadores. Microsoft Store nos proporciona únicamente una respuesta de "licencia válida" o "licencia inválida" sin revelar información del dispositivo.

Datos de Recordatorios y Calendario:

  • Tus recordatorios programados
  • Notas del calendario
  • Horarios y fechas
  • Títulos y descripciones

TODO permanece en tu equipo local.

4. Almacenamiento de Datos

Datos Locales Únicamente

Toda la información que configures en Ninja Reminder Control se almacena exclusivamente en tu dispositivo local en:

%LOCALAPPDATA%\NinjaReminderControl\

Archivos que se crean localmente:

  • Recordatorios y notas del calendario
  • Eventos recurrentes
  • Preferencias de usuario
  • Idioma seleccionado
  • Estilo de notificación

Estos archivos:

  • Solo existen en tu computadora
  • NO se sincronizan con ningún servidor
  • NO son accesibles para el desarrollador
  • Puedes eliminarlos en cualquier momento

5. Comunicaciones con Servidores Externos

Sin Conexión a Internet Requerida

Ninja Reminder Control NO requiere conexión a internet para funcionar y NO se comunica con ningún servidor externo.

Excepciones:

1. Microsoft Store:

  • Cuando descargas o actualizas la aplicación desde Microsoft Store
  • Para validar tu licencia o periodo de prueba
  • Gestionado por Microsoft

2. Actualizaciones (Opcional):

  • Solo si eliges buscar actualizaciones manualmente
  • NO se envía información personal, solo la versión actual de la app

NO HAY:

  • Servicios de análisis de terceros
  • Servicios de publicidad
  • Rastreadores
  • Cookies

6. Información de Pago

Si decides comprar Ninja Reminder Control:

  • Microsoft Store procesa todo el pago
  • Microsoft Store almacena tu información de pago
  • Microsoft Store identifica tu dispositivo y Microsoft Account
  • Nosotros NO tenemos acceso a:
  • Tu información de pago
  • Tu método de pago
  • Datos bancarios o de tarjeta
  • Historial de compras
  • Identificadores únicos de tu dispositivo
  • Tu Microsoft Account

Solo sabemos:

  • Si tu licencia está activa
  • Si estás en periodo de prueba
  • Fecha de inicio de la prueba

Para más información:

Consulta la Política de Privacidad de Microsoft Store

7. Permisos de la Aplicación

Permisos Mínimos

Ninja Reminder Control solicita únicamente los permisos necesarios para funcionar:

Acceso al sistema de archivos local

  • Para qué: Guardar recordatorios, notas y preferencias
  • Obligatorio: Sí

Notificaciones

  • Para qué: Mostrar recordatorios programados
  • Obligatorio: Sí — es la función principal del programa

Tareas programadas de Windows

  • Para qué: Disparar recordatorios incluso cuando la app está cerrada
  • Obligatorio: Sí

Inicio con Windows

  • Para qué: Recuperar recordatorios perdidos al encender la PC
  • Obligatorio: No, pero recomendado

Cliente de internet (internetClient)

  • Para qué: Validar la licencia con Microsoft Store
  • Obligatorio: Sí, solo para Microsoft Store

NO solicitamos:

  • Acceso a cámara o micrófono
  • Acceso a contactos
  • Acceso a ubicación
  • Acceso a fotos/videos

8. Uso de la Información

Como NO recopilamos ningún dato, NO hay uso de información personal.

Los únicos datos que existen son:

  • Configuraciones locales en tu dispositivo
  • Estado de licencia (gestionado por Microsoft Store)

9. Compartir Información con Terceros

NO Compartimos Datos

NO hay recopilación de datos.

No compartimos, vendemos o alquilamos:

  • Datos personales
  • Datos de uso
  • Información de recordatorios
  • Ninguna información

10. Seguridad de los Datos

Protección Local

Dado que todos tus datos están almacenados localmente en tu dispositivo:

  • Tú eres responsable de la seguridad física de tu computadora
  • Windows gestiona los permisos de acceso a archivos
  • Recomendamos:
  • Usar contraseña en tu cuenta de Windows
  • Mantener Windows actualizado
  • Usar antivirus confiable

Por nuestra parte:

  • NO almacenamos datos en servidores
  • NO hay riesgo de filtración de datos de nuestro lado
  • NO hay brechas de seguridad hacia servidores

11. Derechos del Usuario

Tus Derechos

Como NO recopilamos datos personales, no aplican la mayoría de las regulaciones de privacidad (GDPR, CCPA, etc.), pero respetamos tus derechos:

  • Acceso a tus datos: Todos están en tu dispositivo en %LOCALAPPDATA%\NinjaReminderControl\
  • Eliminar tus datos: Desinstala la app o elimina los datos de la carpeta manualmente
  • Portabilidad: Copia los archivos de configuración a otro dispositivo
  • Oposición: No aplica (no hay recopilación)
  • Rectificación: Modifica directamente los recordatorios desde la interfaz de la app

12. Privacidad de Menores

Ninja Reminder Control es seguro para usuarios de todas las edades porque:

  • NO recopila información personal
  • NO tiene publicidad
  • NO tiene contenido inapropiado
  • NO requiere crear cuentas

Clasificación: Apto para mayores de 3+ años (clasificación de Microsoft Store)

13. Cookies y Tecnologías de Seguimiento

NO Usamos Cookies

Ninja Reminder Control no requiere seguimiento por lo tanto:

  • NO usa cookies
  • NO usa localStorage/sessionStorage
  • NO usa píxeles de seguimiento
