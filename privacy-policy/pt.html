Política de Privacidade - Ninja Reminder Control

Última Atualização: Maio 2026

Versão: 1.0

1. Informações Gerais

Ninja Reminder Control é um aplicativo de calendário e lembretes desenvolvido com o máximo respeito à sua privacidade. Esta política explica quais informações coletamos (ou melhor, quais NÃO coletamos) e como protegemos sua privacidade.

Desenvolvedor: Interactive Solutions

Contato: interactive.sup.solutions@outlook.com

2. Compromisso de Privacidade

Princípio Fundamental:

Ninja Reminder Control NÃO coleta, NÃO armazena, NÃO transmite e NÃO compartilha nenhum dado pessoal.

3. Dados que NÃO Coletamos

Este aplicativo NÃO coleta nenhum dos seguintes dados:

Dados Pessoais:

  • Nome, sobrenome, endereço
  • E-mail
  • Número de telefone
  • Data de nascimento
  • Localização geográfica
  • Endereço IP

Dados de Uso:

  • Estatísticas de uso do aplicativo
  • Telemetria ou análise de comportamento
  • Horários de uso
  • Frequência de uso

Dados Técnicos:

  • Informações do dispositivo
  • Sistema operacional
  • Especificações de hardware
  • Identificadores únicos do dispositivo
  • Logs de erros ou diagnóstico

Nota sobre identificadores de dispositivo:

A Microsoft Store usa identificadores de dispositivo para gerenciar licenças entre seus dispositivos. No entanto, nós, como desenvolvedores, NÃO temos acesso a esses identificadores. A Microsoft Store nos fornece apenas uma resposta de "licença válida" ou "licença inválida" sem revelar informações do dispositivo.

Dados de Lembretes e Calendário:

  • Seus lembretes programados
  • Notas do calendário
  • Horários e datas
  • Títulos e descrições

TUDO permanece no seu dispositivo local.

4. Armazenamento de Dados

Apenas Dados Locais

Todas as informações que você configura no Ninja Reminder Control são armazenadas exclusivamente no seu dispositivo local em:

%LOCALAPPDATA%\NinjaReminderControl\

Arquivos criados localmente:

  • Lembretes e notas do calendário
  • Eventos recorrentes
  • Preferências do usuário
  • Idioma selecionado
  • Estilo de notificação

Esses arquivos:

  • Só existem no seu computador
  • NÃO são sincronizados com nenhum servidor
  • NÃO são acessíveis ao desenvolvedor
  • Podem ser excluídos a qualquer momento

5. Comunicação com Servidores Externos

Sem Conexão à Internet Necessária

Ninja Reminder Control NÃO requer conexão à internet para funcionar e NÃO se comunica com nenhum servidor externo.

Exceções:

1. Microsoft Store:

  • Quando você baixa ou atualiza o aplicativo pela Microsoft Store
  • Para validar sua licença ou período de teste
  • Gerenciado pela Microsoft

2. Atualizações (Opcional):

  • Apenas se você optar por buscar atualizações manualmente
  • NÃO se envia informação pessoal, apenas a versão atual do app

NÃO HÁ:

  • Serviços de análise de terceiros
  • Serviços de publicidade
  • Rastreadores
  • Cookies

6. Informações de Pagamento

Se você decidir comprar o Ninja Reminder Control:

  • Microsoft Store processa todo o pagamento
  • Microsoft Store armazena sua informação de pagamento
  • Microsoft Store identifica seu dispositivo e Microsoft Account
  • Nós NÃO temos acesso a:
  • Sua informação de pagamento
  • Seu método de pagamento
  • Dados bancários ou de cartão
  • Histórico de compras
  • Identificadores únicos do seu dispositivo
  • Sua Microsoft Account

Apenas sabemos:

  • Se sua licença está ativa
  • Se você está no período de teste
  • Data de início do teste

Para mais informações:

Consulte a Política de Privacidade da Microsoft Store

7. Permissões do Aplicativo

Permissões Mínimas

Ninja Reminder Control solicita apenas as permissões necessárias para funcionar:

Acesso ao sistema de arquivos local

  • Para quê: Salvar lembretes, notas e preferências
  • Obrigatório: Sim

Notificações

  • Para quê: Exibir lembretes programados
  • Obrigatório: Sim — é a função principal do programa

Agendador de Tarefas do Windows

  • Para quê: Disparar lembretes mesmo quando o app está fechado
  • Obrigatório: Sim

Inicialização com o Windows

  • Para quê: Recuperar lembretes perdidos ao ligar o PC
  • Obrigatório: Não, mas recomendado

Cliente de internet (internetClient)

  • Para quê: Validar a licença com a Microsoft Store
  • Obrigatório: Sim, apenas para Microsoft Store

NÃO solicitamos:

  • Acesso à câmera ou microfone
  • Acesso a contatos
  • Acesso à localização
  • Acesso a fotos/vídeos

8. Uso da Informação

Como NÃO coletamos nenhum dado, NÃO há uso de informação pessoal.

Os únicos dados que existem são:

  • Configurações locais no seu dispositivo
  • Status da licença (gerenciado pela Microsoft Store)

9. Compartilhamento com Terceiros

NÃO Compartilhamos Dados

NÃO há coleta de dados.

Não compartilhamos, vendemos ou alugamos:

  • Dados pessoais
  • Dados de uso
  • Informações de lembretes
  • Nenhuma informação

10. Segurança dos Dados

Proteção Local

Como todos os seus dados são armazenados localmente no seu dispositivo:

  • Você é responsável pela segurança física do seu computador
  • O Windows gerencia as permissões de acesso a arquivos
  • Recomendamos:
  • Usar senha em sua conta do Windows
  • Manter o Windows atualizado
  • Usar antivírus confiável

Da nossa parte:

  • NÃO armazenamos dados em servidores
  • NÃO há risco de vazamento do nosso lado
  • NÃO há brechas de segurança contra servidores

11. Direitos do Usuário

Seus Direitos

Como NÃO coletamos dados pessoais, a maioria das regulamentações de privacidade (GDPR, CCPA, etc.) não se aplica, mas respeitamos seus direitos:

  • Acesso aos seus dados: Todos estão no seu dispositivo em %LOCALAPPDATA%\NinjaReminderControl\
  • Excluir seus dados: Desinstale o app ou exclua a pasta manualmente
  • Portabilidade: Copie os arquivos de configuração para outro dispositivo
  • Oposição: Não aplicável (sem coleta)
  • Retificação: Modifique os lembretes diretamente pela interface do app

12. Privacidade de Menores

Ninja Reminder Control é seguro para usuários de todas as idades porque:

  • NÃO coleta informações pessoais
  • NÃO tem publicidade
  • NÃO tem conteúdo impróprio
  • NÃO requer criação de contas

Classificação: Apropriado para 3+ anos (classificação da Microsoft Store)

13. Cookies e Tecnologias de Rastreamento

NÃO Usamos Cookies

Ninja Reminder Control não requer rastreamento, portanto:

  • NÃO usa cookies
  • NÃO usa localStorage/sessionStorage
  • NÃO usa pixels de rastreamento
